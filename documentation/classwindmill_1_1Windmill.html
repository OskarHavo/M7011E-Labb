def getNext (   self,
  timestamp,
  callback = None,
*  callbackargs 
)

This tries to fetch some data from the windmill.

If no data can be retrieved, the windmill will store a callback function in a buffer and call it upon the next update.

Parameters
timestampTimestamp used to determine which samples the client has already fetched before.
callbackOptional callback function used for buffering requests. The callback must accept a list of data and a timestamp.
callbackargsOptional callback args.
Returns
Data in form of a list and a timestamp of the newest sample or nothing at all.
